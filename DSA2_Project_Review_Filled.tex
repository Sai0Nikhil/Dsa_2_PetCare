\sectionhead{2.\ Project Slides}

{\small
\begin{longtable}{|L{4cm}|L{10.5cm}|}
\hline

\rowcolor{labelgray}\textbf{Abstract} &
  PetCare is a Java-based CLI application modelling a veterinary clinic and pet
  health management platform. It manages pet profiles, medical histories,
  appointments, vaccination schedules, and clinic analytics using Trees, Multiway
  Trees, Graphs, Shortest Paths, and Sorting algorithms applied to realistic
  clinic operations. \\\hline

\rowcolor{labelgray}\textbf{Problem Statement} &
  Traditional veterinary systems lack efficient data-structure-backed operations
  for indexing pet records, optimising inter-clinic routes, and scheduling
  appointments. PetCare maps each DSA module to a concrete clinical problem
  within a single cohesive Java CLI system. \\\hline

\rowcolor{labelgray}\textbf{Objectives} &
  \begin{enumerate}[noitemsep,topsep=0pt,leftmargin=*]
    \item BST \& AVL Trees — $O(\log n)$ pet record indexing, traversals,
          complexity analysis.
    \item B-Tree, B+ Tree, Segment Tree, Fenwick Tree — multiway indexing,
          range queries, prefix sums.
    \item BFS, DFS, connected components, MST (Kruskal / Prim) — clinic
          network management.
    \item Dijkstra, Bellman-Ford, Floyd-Warshall, Topological Sort — route
          optimisation and procedure scheduling.
    \item Divide \& Conquer — Merge Sort and Quick Sort for treatment record
          ranking.
    \item \textit{(Planned)} Greedy \& DP — Activity Selection, Knapsack,
          LIS. \cs.
  \end{enumerate} \\\hline

\rowcolor{labelgray}\textbf{Domain / Application Design} &
  \textbf{Domain:} Veterinary Clinic \& Pet Health Management \newline
  \textbf{Architecture:} Java CLI with nested menus (one per DSA module) and a
  startup data-init screen (CSV / Raw mode). \newline
  \textbf{Entities:} Pet, Treatment, Appointment, Vaccination, Clinic \newline
  \textbf{Packages:} petcare.models, petcare.utils, petcare.menu,
  petcare.trees, petcare.graphs, petcare.sorting, petcare.algorithms \\\hline

\rowcolor{labelgray}\textbf{Selected Data Structures} &
  BST, AVL Tree, B-Tree (order $t$), B+ Tree, Segment Tree, Fenwick Tree,
  Graph (Adjacency List), Priority Queue, Arrays \\\hline

\rowcolor{labelgray}\textbf{Programming Language} &
  Java (JDK 11+) \\\hline

\rowcolor{labelgray}\textbf{Software / Tools Used} &
  Eclipse / IntelliJ IDEA / VS Code, Java JDK, CLI, Git \\\hline

\rowcolor{labelgray}\textbf{Sample Input/Output} &
  Yes — CLI menus run on 45 pre-loaded CSV pet records and clinic data.\newline
  \textbf{M1:} AVL insert/search/delete, BST traversals, height comparison.\newline
  \textbf{M2:} B-Tree search, B+ Tree range query, Segment \& Fenwick sums.\newline
  \textbf{M3:} BFS hop-distances, DFS components, Kruskal \& Prim MST.\newline
  \textbf{M4:} Dijkstra, Bellman-Ford, Floyd-Warshall, Topological Sort.\newline
  \textbf{M5:} Merge Sort (by cost), Quick Sort (by frequency). Heap \&
               Radix — \cs.\newline
  \textbf{M6:} \cs. \\\hline

\end{longtable}
